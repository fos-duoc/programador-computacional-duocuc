## Asignaturas

### 1. Desarrollo de Aplicaciones Móviles
Creación de aplicaciones móviles nativas y multiplataforma.

**Competencias:**
- Desarrollo iOS y Android
- Frameworks multiplataforma
- UI/UX móvil
- Integración con APIs
- Publicación en stores

**Tecnologías:**
- React Native / Flutter
- Swift / Kotlin
- Expo
- Firebase
- App Store / Play Store

### 2. Cloud Native I
Diseño de aplicaciones nativas en la nube.

**Competencias:**
- 12-Factor Apps
- Microservicios
- Containers y orquestación
- Service Mesh
- API Gateway

**Tecnologías:**
- Kubernetes
- Istio / Linkerd
- Kong / Ambassador
- Helm Charts
- Docker Compose
