## Asignaturas

### 1. Cloud Native II
Patrones avanzados de arquitectura cloud native.

**Competencias:**
- Serverless architecture
- Event sourcing y CQRS
- Multi-cloud strategies
- Disaster recovery
- Cost optimization

**Tecnologías:**
- AWS Lambda / Azure Functions
- Event Bridge / CloudEvents
- Terraform multi-cloud
- Chaos Engineering
- FinOps tools

### 2. Seguridad y Calidad de Software
Prácticas de seguridad y aseguramiento de calidad.

**Competencias:**
- OWASP Top 10
- Secure coding practices
- Penetration testing
- Code quality metrics
- Compliance (GDPR, SOC2)

**Tecnologías:**
- SonarQube
- OWASP ZAP
- Snyk / Dependabot
- Vault (secrets management)
- Static/Dynamic analysis
