# MCPs Ready for Using

> Available Model Context Protocol (MCP) servers for enhanced AI capabilities

## Available MCPs

| MCP Server | Abilities |
|------------|-----------|
| **snowflake / snowflake-cicd** | Execute SQL queries, manage objects, semantic views |
| **github** | Repos, PRs, issues, commits, code search |
| **gdrive** | Read/write Google Docs, Sheets, Slides |
| **notion** | Create/edit pages, databases, wikis |
| **figma** | Access designs, components, styles |
| **memory** | Persist context across conversations |
| **brave-search** | Web search |
| **fetch** | Retrieve URL contents |
| **firecrawl** | Advanced web scraping, structured extraction |
| **mermaid** | Generate diagrams: ERD, flowchart, sequence, class, Gantt |
| **drawio** | Create/edit Draw.io diagrams programmatically |
| **playwright** | Browser automation, testing, screenshots |
| **puppeteer** | Web scraping, PDF generation, screenshots |
| **sequential-thinking** | Step-by-step reasoning for complex problems |
| **fs** | Read local files in ITSECKPI project folder |

---

## Usage Notes

- MCPs extend Claude's capabilities beyond standard tools
- Each MCP provides specialized functionality for specific domains
- Multiple MCPs can be used together in a single workflow
