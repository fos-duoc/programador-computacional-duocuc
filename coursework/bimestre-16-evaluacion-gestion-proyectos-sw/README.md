# Bimestre 16 - Evaluación y Gestión de Proyectos de Software

## Ingeniería en Desarrollo de Software - DuocUC

Gestión profesional de proyectos de desarrollo de software.

---

## Asignaturas

### 1. Evaluación de Proyectos de Software
Análisis y evaluación de viabilidad de proyectos.

**Competencias:**
- Análisis de factibilidad
- Estimación de costos
- ROI y métricas de negocio
- Análisis de riesgos
- Business cases

**Herramientas:**
- COCOMO II
- Function Points
- Monte Carlo simulation
- Risk matrices
- Financial modeling

### 2. Gestión de Proyectos de Software
Metodologías y herramientas de gestión de proyectos.

**Competencias:**
- Metodologías ágiles (Scrum, Kanban)
- PMI/PMBOK fundamentals
- Gestión de equipos
- Stakeholder management
- Delivery management

**Herramientas:**
- Jira / Azure DevOps
- Confluence
- MS Project
- Notion / Coda
- Slack / Teams
